# 原型与 UI 设计 (UI/UX Mockups)

## 1. 当前状态
- 当前前端项目仍保留 `create-vue` 默认模板，需要替换为 CRM 业务壳。
- 现阶段没有 Figma、Axure 或视觉稿链接，本文件先作为占位与后续同步入口。

## 2. 首批页面基线
- 登录页
- 联系人列表页
- 联系人详情 / 编辑页
- 黑名单页
- 事项列表页
- 数据看板页
- Contact Agent 页

## 3. 交互约束
- PC 端优先，兼顾移动端窄屏可用性。
- 列表页统一支持搜索、筛选、排序和分页。
- 高风险写操作必须通过弹窗二次确认。
- Agent 的解析结果必须先展示给用户确认，再提交真实写请求。
